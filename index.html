Circuitson

Create your own box2d circuit and export it to a json file.

Your browser does not support canvas or whatever... =(

    [1]: select a object type (line, border, circle, rect).
    A straight lines has 2 points.
    A quadratic bezier curve has 3 points.
    And a cubic bezier curve has 4 points.

    [2]: click in canvas to select points.

    [3]: click "Add element" button to add new elements to circuit.

    [4]: click "Download JSON" button to download json file.
